# Markdown Cheatsheet

Reference: Wes Bos

## Tekst utheving o.l.

**Utheving/ bold** eller **Bold** bruk 2x \* eller \_ for å starte og slutt, e.g. \_\_ / \*\*

_Italic_ eller _Italic v2_ bruk en \* eller \_ for å starte og slutt

~~Strikethrough~~ bruk 2x ~ (hold ALT nede og trykk [*/@])
